# Theme Merge Tool

A simple web tool for merging light and dark theme screenshots with a diagonal split effect.

## Features

- Upload two images (light theme and dark theme)
- Adjust the diagonal split angle (-45° to 45°)
- Download the merged result
- Pure HTML/CSS/JavaScript - no dependencies

## Usage

Visit the live demo: [Theme Merge Tool](https://your-username.github.io/merge-image/)

1. Click on the left side of the canvas to upload a light theme image
2. Click on the right side to upload a dark theme image
3. Adjust the angle slider to change the diagonal split
4. Click "Download Image" to save the result

## Local Development

Simply open `index.html` in your browser. No build process required.
