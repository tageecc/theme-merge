## ✨ 功能特性

- **🖼️ 即时预览** - 上传图片后立即显示，无需等待
- **🎯 对角线分割效果** - 专业的主题对比展示，角度可调
- **📐 角度控制** - 精确调节分割角度（-45° 到 45°）
- **📱 完全响应式** - 在桌面和移动设备上都能完美运行
- **💾 高质量导出** - 下载保持原始分辨率的合并图片
- **⚡ 零依赖** - 纯 HTML/CSS/JavaScript，完全在浏览器中运行
- **🔒 隐私优先** - 所有处理都在本地进行，图片不会上传到服务器
- **🎨 点击上传** - 直观的交互设计 - 点击左侧上传亮色主题，右侧上传暗色主题

## 🚀 快速开始

### 在线版本（推荐）

直接访问 **[https://tageecc.github.io/merge-image/](https://tageecc.github.io/merge-image/)** 即可立即使用！

### 本地开发

1. 克隆仓库
```bash
git clone https://github.com/tageecc/merge-image.git
```

2. 在浏览器中打开 `index.html`
```bash
cd merge-image
open index.html  # macOS
# 或者直接双击 index.html 文件
```

就是这么简单！无需构建过程，无需安装依赖。

## 📖 使用说明

1. **上传第一张图片**
   - 点击画布的左侧区域上传亮色主题截图
   - 图片会立即显示预览

2. **上传第二张图片**
   - 点击画布的右侧区域上传暗色主题截图
   - 对角线合并效果会自动应用

3. **调整角度**
   - 使用滑块调整对角线分割的角度
   - 实时预览调整效果

4. **下载结果**
   - 点击"下载图片"按钮
   - 合并后的图片会以当前日期命名并保存
